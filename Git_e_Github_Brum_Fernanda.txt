1.Quem inventou o sistema de controle de versão Git e por quê?
Linus Torvalds. O mesmo inventou a versão Git pois estava insatisfeito com a mudança de licença de utilização do BitKeeper(sistema de versionamento utilizado por ele na época). Então, decidiu criar o Git para gerenciar melhor as mudanças que ocorrem no código-fonte de um projeto.

2.Quem atualmente é o dono do Github e por quê?
No dia 04 junho de 2018, a Microsoft adquiriu o GitHub por U$ 7,5 bilhões em ações, com o objetivo de fortalecer o compromisso com a liberdade, a abertura e a inovação dos desenvolvedores.

3.Existe outra maneira, além do terminal, de trabalhar com o Github?
Sim, é possível usar o programa desktop do Github, plataforma Web ou o próprio terminal do VS Code.
